# MEDI 110   Final Website Project  
### Author:  Nico Ramos  
### Student: Vancouver Island University

## Overview  
This project is a five-page static website created for the MEDI 110 (Introduction to Media production) final assignment.  
The purpose of the website is to use photography, place-based storytelling, and basic web design to explore how Vancouver and Nanaimo influence my creative process as a student and emerging developer.

The site blends visual storytelling, travel photography, reflections, and simple HTML/CSS layout to communicate how place and environment shape creativity.

## Design Principles

- **Minimal, clean layout** to ensure the photography stands out  
- **Consistent color palette** inspired by West Coast environments  
- **Large, prominent images** used as primary storytelling elements  
- **Sticky navigation bar** for easy page-to-page movement  
- **Readable typography and spacing** for a comfortable viewing experience  

---

## Technologies Used

- HTML5  
- CSS3  
- Custom photography  with iphone 16 
- No JavaScript or external frameworks  
- Fully static, hand-coded structure

## Content Creation

All photos used in the project were taken personally in:  
- Nanaimo Harbour  
- Downtown Vancouver  
- Vancouver Island University campus  

These locations were chosen to explore contrasts between nature and city, calm and movement, and student life and travel.

This project allowed me to combine web design with digital media concepts.

It also helped connect my technical interests in web development with creative expression.
